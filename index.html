A little about me.

I'm a student coder on her way to front-end web developer and beyond. I found my way into tech during the pandemic, when my work as a professional actor began to dry up as theaters and film sets were shutting down. Looking for a new challenge, I turned to learning code and quickly fell in love with the creative potential that work as a web developer offered. After I saw my first hand-coded landing page light up a computer screen, I was hooked. I soaked up HTML5, CSS3, and Git in a flash and have recently dived headlong into responsive web design.

I might still be learning, but I'm already building sites with my new skills. Wanna see for yourself?
